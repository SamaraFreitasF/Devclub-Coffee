# Devclub-Coffee
Um site fictício de cafeteria desenvolvido em HTML e CSS, inspirado no universo da programação. O projeto simula uma página moderna e responsiva para uma coffee shop, destacando produtos, avaliações de clientes e localização.

📌 Funcionalidades

Página inicial.

Seção Sobre Nós com descrição da cafeteria.

Menu de Produtos exibindo cafés com preços e botão de adicionar ao carrinho.

Avaliações de Clientes com depoimentos e sistema de estrelas.

Endereço integrado ao Google Maps.

Rodapé com links de redes sociais.

🛠️ Tecnologias utilizadas

HTML5 para a estrutura do site.

CSS3 para estilização e responsividade.

Google Fonts (Roboto) para tipografia.

Icons8 para ícones de interface.
